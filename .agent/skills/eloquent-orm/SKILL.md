# Eloquent ORM Skill

## Key Concepts
- **Relationships**: Properly define `hasMany`, `belongsTo`, `belongsToMany`.
- **Eager Loading**: Use `with()` to prevent N+1 query problems.
- **Scopes**: Use Local Scopes for reusable query logic.

## Security
- Always use mass assignment protection (`$fillable` or `$guarded`).
- Sanitize user input before using it in raw queries.
