# Preline UI Skill

Preline UI provides pre-built components for Tailwind CSS.

## Best Practices
- **Customization**: Use Tailwind config to extend colors and fonts.
- **Components**: Check [Preline Documentation](https://preline.co/) for copy-paste components.
- **JavaScript**: Ensure Preline's JS is initialized in `resources/js/app.js`.

## Example
```html
<button type="button" class="py-3 px-4 inline-flex items-center gap-x-2 text-sm font-semibold rounded-lg border border-transparent bg-blue-600 text-white hover:bg-blue-700 disabled:opacity-50 disabled:pointer-events-none dark:focus:outline-none dark:focus:ring-1 dark:focus:ring-gray-600">
  Button
</button>
```
