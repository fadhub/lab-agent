# Spatie Packages Skill

Spatie produces essential packages for Laravel.

## Core Packages
- **laravel-permission**: Role and permission management.
- **laravel-medialibrary**: Handle file uploads and image conversions.
- **laravel-query-builder**: Filter and sort Eloquent queries via URL.

## Usage Patterns
- Always follow the `HasRoles` or `InteractsWithMedia` trait patterns as documented by Spatie.
- Use migrations provided by the packages for schema setup.
